# Multimodal Mafia

To run, install the requirements, then run any of the `.bat` scripts.

## Commands

next  
- Proceed to next phase

kill X  
- Kill / lynch player X

save X  
- Doctor saves player X

check X  
- Detective checks player X

repeat  
- Repeat roles to all

repeat X  
- Repeat role to player X

status  
- Show game state & roles

reset  
- Restart with new roles

help  
- Show available commands

quit  
- Exit game
